# Mini PHP Drive

เว็บแชร์ไฟล์ขนาดเล็กที่รันบน PHP + Apache ผ่าน Docker Compose พร้อมระบบล็อกอินด้วยรหัสผ่านเดียวและการอัปโหลด/ดาวน์โหลดไฟล์

## สิ่งที่ต้องมี

- Docker และ Docker Compose

## วิธีรันโครงการ

1) สร้างและรันคอนเทนเนอร์`docker-compose up -d`
2) เปิดใช้งานที่ `http://localhost:8080`

หยุดบริการ: `docker-compose down`
ข้อมูลไฟล์ทั้งหมดถูกเก็บบนโฮสต์ที่โฟลเดอร์ `app/uploads` (แมปเป็น volume ไปยังคอนเทนเนอร์)

## การใช้งาน

- ล็อกอินด้วยรหัสผ่านจาก `app/config.php` (ค่าเริ่มต้น `123456`) หากต้องการเปลี่ยนให้แก้ไขไฟล์แล้วรัน `docker-compose up -d --build`
- อัปโหลดไฟล์/โฟลเดอร์: ลากมาวางในกล่อง หรือกดเลือกไฟล์/โฟลเดอร์ (รองรับ multiple และโครงสร้างโฟลเดอร์)
- สร้างโฟลเดอร์: พิมพ์ชื่อแล้วกดปุ่ม `+`
- ดาวน์โหลด/ลบ: ใช้ลิงก์ในตารางรายการไฟล์
- นำทางเข้าโฟลเดอร์ด้วยการคลิกชื่อโฟลเดอร์ และใช้ลิงก์ “ย้อนกลับ” ในตารางเพื่อกลับระดับบน
- หน้าจอ UI คล้าย Google Drive (ธีมเข้ม) มีช่องค้นหา, การ์ดไฟล์, แผงรายละเอียดด้านขวา, drag & drop สำหรับไฟล์/โฟลเดอร์

## รายละเอียดบริการ

- service `php-drive` สร้างจาก `Dockerfile` (ฐาน `php:8.2-apache` ติดตั้งส่วนขยาย `mysqli`)
- เปิดพอร์ต 8080 ของเครื่องไปยังพอร์ต 80 ในคอนเทนเนอร์
- แชร์โฟลเดอร์ `./app/uploads` ไปยัง `/var/www/html/uploads` และตั้งค่าให้ restart เว้นแต่ถูกสั่งหยุด

## หมายเหตุด้านความปลอดภัย

- ใช้รหัสผ่านเดียวและเก็บแบบ plain text; เหมาะสำหรับใช้งานส่วนตัวหรือเครือข่ายที่ไว้ใจได้เท่านั้น
